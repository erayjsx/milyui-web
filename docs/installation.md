# Installation

Install

```Shell
npm install @erayjs/milyui
```

For CDN

```Html
<script src="https://unpkg.com/@erayjs/milyui"></script>
```

```Html
<script src="https://cdn.jsdelivr.net/gh/erayjsx/milyui@latest/dist/bundle.js"></script>
```
